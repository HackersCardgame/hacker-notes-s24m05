filter returns a subset of the collection, with only those elements for which the function you gave it returns true. So [1, 2, 3, 4].filter(isEven) returns [2, 4]


map maps each value in a collection to a new value/type. So [1, 2, 3, 4].map(squared) returns [1, 4, 9, 16]

reduce reduces a whole collection to a single value, given some seed as a starting point and a function to operate upon each value and the previous result. So [1, 2, 3, 4].reduce(0, add) returns 10

These can be easily combined, so [1, 2, 3, 4].lazy.filter(isOdd).map(squared).map(toString).reduce("", concatenate) returns "19". If you do what I showed here and have it lazily evaluated, the performance is the same as doing all these in a traditional for loop
