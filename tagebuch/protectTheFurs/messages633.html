The issue with php isn't the language itself, it's the ecosystem and its terrible deployment tools. The language invites some bad habits but the way you deploy PHP is a core issue.

PHP is at its core the last true vestige of a very old and very outdated way of deploying software: CGI.

When web applications became bigger, we began growing tired of either forwarding everything inside cgi-bin or finding a way to hide the truth.
